Security Vulnerability Analysis based on Cloud Platform with Intel IA Architecture

 

General Principals

--------------------------

With the advance of computer technology and computer network, more and more security concerns are emerging, such as the public's privacy leakage and online system's breakdown etc. The network security status are facing growing threats. Therefore, it's necessary to utilize collaborative network security and cloud-based platform for network security in teaching courses.

 

Topics

--------------------------

 

1.  Introduction

     Cloud Platform with Intel IA Architecture.

2.  Network Security and Internet Security

     Cyberspace, SSA security situational awareness, Collaborative Network Security Management system

3.  Network Security Principles and Techniques in Mobile Platform

     Threats, Attacks and Defense.

4.  Named Data Networking - Future Internet architecture

     Security and Privacy issues of Named Data Networking.
